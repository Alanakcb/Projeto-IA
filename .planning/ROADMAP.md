# Roadmap

## Milestone 1: Classificador de Óleo

### Phase 1: Implementar Perceptron para Classificação de Óleo
- **Goal:** Construir o classificador Perceptron do zero com NumPy e gerar gráficos de treinamento.
- **Depends on:** None
- **Plans:**
  - Plan 1: Preparar os dados de treinamento e inicializar os parâmetros (viés, pesos aleatórios).
  - Plan 2: Implementar o loop de treinamento (ajuste de pesos) e critério de parada.
  - Plan 3: Implementar a geração de gráficos com Matplotlib por época.
